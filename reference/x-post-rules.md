# X投稿ルール

`/post-as-clove-user` および `scripts/post_to_x.py` が遵守する投稿の機械的ルール。
**ペルソナ・例文より優先される強い制約**。AIにはこのルールを必ず守らせる。

---

## 文字数

- 基本: **50〜140字**
- 短すぎ（30字未満）も長すぎ（200字超）もNG
- 140字を超える場合は短く言い換える（リプライ・スレッド化しない）

---

## ハッシュタグ

- 1投稿につき **0〜2個** まで（3個以上は禁止）
- 使ってよいタグ:
  - `#ポケカ`
  - `#オリパ`
  - `#Clove`
  - `#TCG`
  - `#ポケモンカード`
- 上記以外の独自タグは作らない
- ハッシュタグなしの投稿も半数程度は出していい（自然さ優先）

---

## 絵文字

- 1投稿につき **0〜2個** まで
- 連続使用禁止（`🔥🔥🔥` のような連打NG）
- 使ってよいもの: 🎴 ✨ 💭 🤔 😅 ☕ など落ち着いた系
- 使わないもの: 🔥💯💸 のような煽り系、ハート系（💖❤）の多用

---

## 句読点・記号

- 「！」「!!」「!!!」の多用禁止: 1投稿につき1個まで
- 「？」は普通に使ってよい
- 「…」は1投稿につき1個まで
- 全角・半角の混在は許容（自然な範囲）

---

## 禁止事項（絶対NG）

| 種類 | 例 |
|------|------|
| 過度な煽り・断定 | 「絶対当たる」「損はしない」「神すぎる」「ヤバすぎ」 |
| 呼びかけ調 | 「みんなも引こう」「ぜひ試して」「やってみて！」 |
| まとめサイト調 | 「【速報】」「【朗報】」「〜な件について」 |
| 他社・他者言及 | 特定の他社サービス名、他Xアカウントへの言及・@メンション |
| 比較dis | 「Aより良い」「他より当たる」など比較貶し |
| 個人情報 | 住所・本名・勤務先・年齢の特定など |
| センシティブ | 政治・宗教・差別・社会問題への意見 |
| ステマ感全開 | 毎投稿Clove宣伝、不自然なポジティブ連発 |

---

## 重複・連投回避

- 直近20投稿の **話題・表現・出だしの言葉** が被らないように
- 同じカード名・同じネタを2日以内に再使用しない
- 文体（です・ます調 / だ・である調 / 口語）も連続させすぎない
- 「〜だな」「〜かも」など語尾も連続3回までを目安

---

## Clove言及の頻度

- 10投稿のうち **3〜4回まで** にClove言及を抑える
- 残りはポケカ・TCGの一般話題、または日常雑談
- Clove言及時も「使ってる側」目線を崩さない

---

## 時間帯による話題のニュアンス（任意・あれば反映）

| 時間帯 | 話題のヒント |
|---|---|
| 朝（8〜10時） | おはよう系、通勤、寝起きのぼやき、今日の予定 |
| 昼（11〜14時） | 昼休み、ランチ、午後の気だるさ |
| 夕方（15〜18時） | 仕事終わり、帰宅、夕飯前 |
| 夜（19〜23時） | リラックス、開封タイム、明日の準備 |

固定する必要はない（自由に外してOK）が、深夜帯の「徹夜」「寝てない」系は出さない（運用時間外）。

---

## 投稿後の検証ルール（人間用）

最初の1週間は以下を毎日チェック:

- [ ] Bot臭く見えていないか（タイムライン一覧で見たとき）
- [ ] 同じパターンの繰り返しがないか
- [ ] ハッシュタグ・絵文字の使用頻度が適切か
- [ ] ステマ臭くなっていないか（Clove言及バランス）

悪い例が出たら → このファイルか `x-post-examples.md` の「NG例」に追記して次回反映。

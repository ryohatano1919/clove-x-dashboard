# X投稿のお手本集

`/post-as-clove-user` および `scripts/post_to_x.py` が参照する投稿例。AIはここの例を「目指す品質」として参考にする。

**運用しながら育てるファイル**。反応が良かった投稿、自分が見て「これは自然」と思った投稿を追記していく。

---

## 形式

各例は以下のフォーマット:

```
### [ジャンル名] サンプルN
[投稿本文をそのまま]
[空行]
（コメント: なぜ良いか / 学習ポイント）
```

## NG例（こう書かない）

❌ 「Cloveのオリパ最高すぎ！！！みんなも引こう！！！🔥🔥🔥」
→ テンション高すぎ・呼びかけ・絵文字過多・ステマ臭

❌ 「【速報】Cloveの新オリパが神すぎる件について」
→ まとめサイト調・煽り

❌ 「リザードン狙うなら絶対Clove！損はさせません！」
→ 断定・煽り・宣伝臭

❌ 「他社のオリパよりCloveの方が当たる気がする」
→ 比較disはNG

## 育てる時の追記方法

1. 良い投稿を見つけたら、適切なジャンル下に **サンプルN** として追記（番号は連番）
2. コメント欄に「何が良いか」を1〜2行で書く
3. ジャンルが既存にない場合は、新しいセクションを追加してOK
4. **NG例**も育てる: 出てしまった悪い投稿は「NG例」に追加し、なぜダメかコメント
